# 🚀 GitHub User Explorer

A simple and responsive web application built with **HTML, CSS, and Vanilla JavaScript** that fetches GitHub users using the GitHub REST API and displays their profile information.

---

## 📸 Home Page

![Home Page](./Home.png)

## 👤 Details Page

![Details Page](.Details.png)

### Home Page

* Displays a list of GitHub users.
* Shows profile image and username.
* Link to visit the user's GitHub profile.

### Details Page

* Displays:

  * Avatar
  * Name
  * Username
  * Bio
  * Followers
  * Following
  * Public Repositories
  * Location

---

## ✨ Features

* Fetch GitHub users using Fetch API
* Display user cards dynamically
* Click on a profile image to view detailed information
* Separate details page
* Responsive layout using Flexbox
* Hover animations
* Error handling using try...catch
* Clean UI

---

## 🛠️ Technologies Used

2. Open the project folder.

3. Open `index.html` in your browser.
